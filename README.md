# WELLNESS-ANAYSIS-PROJECT
This dataset analyse how sleep patterns, mobile usage, and stress levels interact across 15,000 records to identify wellness risks and opportunities. The purpose of this dataset is to analyse correlations; how mobile phone usage before bedtime affects sleep quality and whether poor sleep pattern is associated with stress.
Screen time impact (Low: stress 3.42 → High: stress 9.34), occupation rankings, pre-sleep phone use, poor sleep rates by job, and high-risk population profile.
Heavy mobile usage before bedtime strongly correlates with reduced sleep quality.
 Average stress of 6.98/10, only 6.51 hrs sleep, 10.1% high-risk population.
 A significant portion of users are not meeting recommended sleep hours, which is linked to higher stress.
